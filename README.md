# 🚀 Knowledge Hub Platform

Hệ thống quản lý kiến thức lập trình đa ngôn ngữ, hỗ trợ thực thi code trực tiếp.

## 📌 Mục tiêu dự án
Xây dựng nền tảng cho phép lưu trữ và thực thi các kiến thức lập trình (Code Snippets, Cấu trúc dữ liệu, Giải thuật) với kiến trúc Microservices/Modular.

## 🛠 Tech Stack
- **Frontend:** Next.js, React, Tailwind CSS
- **Backend:** Go (Golang)
- **Database:** MySQL (Metadata), MongoDB (Content)
- **Infrastructure:** Docker, CI/CD

## 📚 Tài liệu chi tiết
Tất cả thiết kế, kiến trúc và nhật ký phát triển được ghi chép tại [Wiki của dự án](https://github.com/truongnguyen-kt/knowledge-hub-platform/wiki).
